# Homepage — CSS Assignment

## Description

A responsive homepage built with HTML and CSS as part of the ALU/ALX Software Engineering program. This project focuses on implementing a pixel-perfect layout from a Figma design using only HTML and CSS — no JavaScript frameworks or CSS libraries.

## Learning Objectives

- Translate a Figma design into a working webpage
- Use CSS layout techniques (Flexbox and/or Grid)
- Apply responsive design principles
- Work with typography, spacing, and color systems
- Structure semantic HTML

## Project Structure

```
homepage/
├── index.html
├── styles/
│   └── style.css
├── images/
│   └── (assets used in the page)
└── README.md
```

## Requirements

- HTML5 and CSS3 only — no external CSS frameworks (e.g. Bootstrap, Tailwind)
- No JavaScript
- Page must match the Figma design as closely as possible
- Must be responsive across common screen sizes
